# MicroLex English (MLE)
### *A Constructed English Dialect with a 1–4 Letter Word Limit*

> **Motto:** *"Say big. Use wee."*

---

# Overview

**MicroLex English (MLE)** is a constructed dialect of English where every word used in speech or writing must contain only one (1), two (2), three (3), or four (4) letters. Any word that contains five (5) or more letters is strictly forbidden.

Unlike standard English, the focus of MLE is not on changing grammar or pronunciation, but on vocabulary. The aim of the dialect is to prove that clear, rich, and complex ideas can still be communicated effectively using only very short words. Speakers and writers must replace long words with shorter synonyms, abbreviate them, describe the concept, or rewrite the sentence entirely.

---

# Core Principle & Scope

Every single written and spoken word must obey the length rule:

| Letters | Status | Examples |
| :--- | :--- | :--- |
| **1** | ✅ Allowed | A, I |
| **2** | ✅ Allowed | do, go, to, we |
| **3** | ✅ Allowed | cat, run, sky, new |
| **4** | ✅ Allowed | bird, play, good, read |
| **5+** | ❌ Forbidden | Never used |

This rule applies universally to **all** parts of speech and language elements, including:
*   Nouns, Pronouns, Verbs, Adjectives, Adverbs
*   Articles, Prepositions, Conjunctions, Interjections
*   Proper nouns, Personal names, Place names
*   Days, Months, Titles, Acronyms

Nothing is exempt.

---

# Grammar

Grammar remains nearly identical to standard English; only the vocabulary is limited. 

Sentence order remains: **Subject → Verb → Object**

Verb tenses still exist but must use legal words. 
*   *Normal:* I completed the work.
*   *MLE:* I did end the job.

---

# Vocabulary Rules (Handling Long Words)

If a word has five or more letters, one of the following methods must be used to replace it.

## Rule 1 — Prefer a Short Synonym
Whenever a shorter synonym exists, it must be used.

| Normal (Forbidden) | MLE (Allowed) |
| :--- | :--- |
| large | big |
| little | tiny |
| child / children | kid / kids |
| father | dad |
| mother | mom |
| begin / start | open / set off |
| finish | end |
| purchase | buy |
| obtain | get |
| create | make |
| reside | live |
| around | by |
| because | for |
| before | ere |

## Rule 2 — Split the Idea / Rewrite the Sentence
If no legal synonym exists, rewrite the sentence to convey the same meaning.

*   *Normal:* Yesterday I purchased vegetables.
*   *MLE:* Last day I did buy veg.

## Rule 3 — Describe or Explain the Object
If one word cannot be used, describe what it is using short words.

| Normal (Forbidden) | MLE (Allowed) |
| :--- | :--- |
| elephant | big zoo pet / big zoo pup |
| airplane | sky jet |
| computer | PC |
| helicopter | air cab |
| telescope | sky aid |
| notebook | note pad |

# Category Rules

## Verb Conversion
Many long verbs become helper verbs.

| Normal (Forbidden) | MLE (Allowed) |
| :--- | :--- |
| completed / finished | did end |
| arrived | got in |
| departed | set off |
| discovered | did see |
| constructed / created | made |
| developed | did grow |
| decided | did opt |
| entered | went in |

## Time Expressions

| Normal (Forbidden) | MLE (Allowed) |
| :--- | :--- |
| yesterday | last day |
| tomorrow | next day |
| morning | day rise / day start |
| afternoon | day mid |
| evening | day end |
| tonight | this eve |

## Numbers
Digits are always allowed (e.g., `1`, `12`, `27`, `365`, `999`, `2026`, `999999`). 
Written numbers must obey the letter count rule.

| Word | Status |
| :--- | :--- |
| one, two, six, ten, zero | ✅ Allowed |
| five, nine | ✅ Allowed |
| three, seven, eight, eleven | ❌ Forbidden |

## Proper Names
Names longer than four letters should be shortened using a short form, an ID, initials, or a nickname.

| Normal (Forbidden) | MLE (Allowed) |
| :--- | :--- |
| Robert | Rob |
| Samuel | Sam |
| Jennifer | Jen |
| Michael | Mik / Mike |
| William | Will |
| Elizabeth | Liz |

## Place Names
Long place names must be cut or abbreviated.

| Normal (Forbidden) | MLE (Allowed) |
| :--- | :--- |
| England | Eng |
| Kolkata | Kol |
| America | USA |
| Germany | Ger |
| Australia | Aus |
| Scotland | Scot |

## Acronyms
Acronyms are highly encouraged as long as they are four letters or fewer (e.g., USA, UK, EU, AI, PC, CPU, RAM, SSD, GPS, HTML).

# Formatting and Writing Rules

*   **Punctuation:** Normal English punctuation is fully preserved (`. , ! ? : ; " ' () []`).
*   **Capitalization:** Standard English capitalization rules apply. Sentence beginnings, proper nouns, and acronyms remain capitalized.
*   **Word Formation:** Compound words may be split if necessary (e.g., instead of `football` use `foot ball`; instead of `sunlight` use `sun ray`).
*   **Borrowed Words:** Allowed only if they obey the length rule (Allowed: `cafe`, `yoga`, `guru`, `dojo`. Forbidden: `karaoke`, `tsunami`, `spaghetti`).
*   **Technical Writing:** Technical concepts should be simplified. 
    *   `algorithm` → `step set`
    *   `database` → `data bin`
    *   `network` → `net`
    *   `compiler` → `code tool`

# Word Lists

## Common "Good" Words (1-4 Letters)
*   **Pronouns/Helpers:** I, a, an, am, is, are, was, has, had, can, may, will, must, all, any, few
*   **Adjectives:** new, old, good, best, real
*   **Animals:** ant, bee, bear, wolf, lion, frog, deer, crab, crow, swan, duck, goat, cat, bird, fish
*   **Nature/Things:** sky, sea, rain, wind, rock, hill, tree, fern, leaf, sand, snow, fire, star, moon, sun, road, home, book, note, page, code, math
*   **Actions:** walk, jump, sing, read, draw, make, send, keep, hold, pull, push, hear, play, run, cook, talk, find, know, love, help, need, work, take, give, grow

## Common "Bad" Words (5+ Letters)
*Forbidden to use:* people, computer, language, beautiful, between, another, without, important, history, science, writing, dictionary, sentence, because, different.

# Example Sentences & Texts

**Normal:** Yesterday my father purchased another computer because the previous machine failed.
**MLE:** Last day my dad did buy new PC for the old one did die.

**Normal:** Scientists discovered a beautiful galaxy beyond another star.
**MLE:** Lab men did see one fair sky sea far off one new sun.

**Normal:** The children enjoyed playing football after school.
**MLE:** The kids did joy play foot ball aft sch.

**Normal:** Artificial Intelligence is transforming modern society.
**MLE:** AI can aid all way of man now.

**Normal:** Yesterday our teacher explained a difficult language exercise before lunch.
**MLE:** Last day our sir did say one hard Eng act ere eat.

**Normal:** The small village became famous because people created beautiful wooden houses beside the river.
**MLE:** The wee ham did get fame for all did make nice wood hut by the sea.

### Example Conversation

**Normal:** Where are you going tomorrow?
**MLE:** Who go you next day?

**Normal:** I purchased a computer yesterday.
**MLE:** I did buy a PC last day.

**Normal:** Please explain this difficult problem.
**MLE:** Can you say how this hard ask can end?

# Stories

## A Sax way

This is one old way of Eng. It was use by the folk of old. It had its own set of law and art. The way did not use all the same old set now.

One key bit was its own way for say and ink. Man and boy did use it day by day. Far and sea did aid its use.

Now few can use all of it, but its art can aid us see how Eng did grow. Old ink, old law, and old lay all show its way. Its use did end for day use, but its mark did not.

Its aid can let us see how one way can add to all our own way of Eng.

## A Kent way

The Kent way is an old way of Eng. It was use in the far south of the land. Men and all did say its own set of old and new art.

One key bit was how all did say and use old air and old tone. Some did say "ich" for "I". All had its own way for talk and for day to day use.

This way did keep old Eng way for a long era. New way did win in the end, but the old way did not die all at one.

Now few can say it as all did in old day. But old ink and old map can aid us see how it did work. It can aid us see how Eng did grow and how one land can use its own way of talk.

The Kent way is one part of the long tale of Eng. Its old way can aid all who wish to see how our own way of talk came to be.

# Uses, Advantages & Challenges

## Uses
MicroLex English can be used for:
*   Fun word games and puzzle books
*   Brain training and cognitive exercises
*   Writing contests and minimalist writing
*   School language activities
*   Coding comments
*   Secret messages

## Advantages
*   Encourages concise writing and makes writers think before choosing words.
*   Expands vocabulary and develops paraphrasing skills.
*   Improves creativity and produces unique writing styles.
*   Demonstrates that simple language can express complex ideas.

## Challenges
*   Technical subjects become difficult to explain.
*   Many common words require rewriting, and some ideas require longer, multi-word explanations.
*   Proper names often need abbreviations, and formal writing/poetry may lose detail.
*   Writing speed may decrease as finding short replacements can take time.

# Summary

MicroLex English is a rule-based dialect of English in which **every individual word must contain only one to four letters**. Long words are replaced by short synonyms, abbreviations, descriptions, or rewritten expressions while preserving the original meaning. The dialect emphasizes simplicity, creativity, and efficient communication under strict lexical constraints.
