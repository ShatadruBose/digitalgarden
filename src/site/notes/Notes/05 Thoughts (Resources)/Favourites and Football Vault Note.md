# Favourites and Football Vault Note

Overview

This note is a personal favourites map. It records the things I like most in different categories, and it also builds a football vault where each player, team, league, or role can unlock a nation, club, or football identity for my system.

The idea is simple:

- One favourite = one anchor
- One football name = one nation, club, or role
- One vault = one structured world of preferences

This makes the note both personal and usable. It is not just a list. It is a classification system for my interests, my football world, and the way I organise knowledge.
